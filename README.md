#### Instalação de pacotes e dependências
```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

### Compilação e abertura de rede para rodar localmente

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```

### Não é necessário token de API para a comunicação

```
A API utilizada neste projeto foi a DummyJSON

https://dummyjson.com/
```
